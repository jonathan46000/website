Scan Line Compositor

When I first started this I knew nothing about low level graphics manipulation and how to build a graphics system. By the time I was done it wasn't the prettiest work but it worked well, and now I can do much better. This same year I was working a lot with XLib and XCB as well and this was much lower level than that. The badge project this was done on

is a project that HackRVA does for RVASec every year. HackRVA is a makerspace here that I am a member of. This badge for the RVASec conference is running on a PIC32 Microprocessor. All of the code is written in C and compiled with the XC32 compiler.
