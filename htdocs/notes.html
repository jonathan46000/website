The server caught fire! Long live the server!

October 22, 2017

The server caught fire in the PSU, and killed the hard drives along with it. Luckily the motherboard survived and full backup has been restored to the new drives and a new PSU installed (As evidenced by the existence of the web page). The new power supply is of decidedly higher quality and is not likely to repeat the burnage event.

Iceland, RVASec, LUG

July 4, 2017

Many things have happened since the last time I blogged on here. I have visited iceland, attended RVASec 2017 as a developer for the RVASec badges, started a new software project and have become the president of the Linux Users Group at the college I attend. First of all, Iceland is a beautiful country and if you ever have the chance you should go there. Similarly RVASec 2017 was a blast and the HackRVA badges were a hit once again. Two of my apps made it into the badge this year. You can find the source code for the entire badge at https://github.com/HackRVA/Harmony-Badge-2017-public and the code to my drawing app for the badge at https://github.com/HackRVA/Harmony-Badge-2017-public/blob/master/firmware/src/badge_apps/udraw.c. I recently started making a simple GUI for wget in Qt. It's not ridiculously robust but it's very simple and provides a couple of nice features that make it convenient for me to use with blackboard. You can find the code for this at https://github.com/jonathan46000/crude-wget-gui. Running a Linux Users Group has tought me a lot. I have never claimed to be an expert at Linux(most linux users would never dare claim to be an expert). I am surprised however at how much you can learn by helping others. This has been a very fulfilling experience for me. Now last, but not least, Happy Independence Day!
