Drawing App

This is a really simple HTML Canvas game written in JavaScript that lets you change colors, brush size, and pick between two different brush shapes(circle and square). This was my first HTML Canvas application and was meant to be very simple so that I could learn how to use JavaScript to create these applications.

Star Trek Side Scroller

This Star Trek based side scroller fan app was an interesting build. I learned a lot about HTML Canvases and event driven JavaScript code. The entire game is written on boiler plate code without using frameworks. All of the controls are functional on the screen but I recommend using the keyboard commands if you don't want to die quickly. The game is fun to play and gets challenging in the 2nd and 3rd levels.
