Crude Wget Gui

This is a project I started so that I could conveniently download lists of items from accounts I needed to log into. It allowed me to copy and paste the links and then download them all. It's not an extremely full featured application but it does what I need it to do. It is written in Qt 5.6 but should compile and run in 5.9.

Qt Calculator

In 2014 I gave an 8 week course at HackRVA on how to program in C++ and use the Qt Framework to build form applications that you could compile anywhere. The last project of the course was a simple calculator application written in Qt4. All of the sources are provided with the code heavily commented. If I can find it at some point in the future there is a power point slide I will provide with the code. Recently updated to be compatible with Qt 5.6.
